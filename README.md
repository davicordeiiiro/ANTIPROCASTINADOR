# ANTIPROCASTINADOR
- **PROPOSTA** 

O antiprocostinador será um software que auxiliará o usuário a organizar suas tarefas Sobre tecnologia, e ter total conrtrole sobre seu progresso.

    - **O programa possui:**
    - Temporizador
    - Cronômetro
    - disparador de notificações
    - relatórios
    - uma interface amigável
    - funcionamento em segundo plano




- **DADOS DE ENTRADA NECESSÁRIOS**

O usuário deverá informar ao programa quais são as suas tarefas em andamento, o que ele planeja estudar no futuro, em quanto tempo ele deve terminar, e qual será o rítimo desajado para executálas.

- **QUAIS SÃO AS RESTRIÇÕES**

O usuário não pode inserir tarefas que estão fora da capacidade do programa.

O programa não tem a capacidade de acessar links de videos ou cursos para exibir o seu desempenho, ele deve inserir manualmente

- **QUAL É O RESULTADO ESPERADO?**

- Que o programa envie noificações para o 
  usuário periodicamente sobre o seu desempenho.

- Que o programa exiba um relatória semanal, mensal e anual.

    - *(futuramente)*
    - que o programa envie tendências do ramo da programação
